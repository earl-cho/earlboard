# SYSTEM_OVERVIEW.md

이 문서는 프로젝트의 전반적인 상태와 에이전트의 컨텍스트를 보존하기 위해 작성되었습니다.

## 1. 개요 (Architecture)
- **프로젝트 명**: Blackboard Researcher (가칭)
- **주요 기능**: 크립토, RWA, 엔터프라이즈 채택, 사이버펑크 철학 관련 데이터 수집 및 AI 인사이트 생성 자동화.
- **기술 스택**: Next.js, Supabase, Python (Crawlers), LLM (Gemini 3.5 Pro).
- **에디터 로직 (`editor_v3.py`)**: 
    - **최소 3개 이상의 소스**가 모였을 때만 통합 리포트 작성.
    - **VIP 소스 가중치 (+20점)**: 매일/한국/서울경제, 파이낸셜뉴스, 디지털애셋, CoinDesk, The Block, WuBlockchain 등 지정된 핵심 매체.
    - **일본 컨텍스트 부스트 (+20점)**: 일본 매체에서 규제(FSA), 공공, 대기업(Sony, NTT 등) 관련 내용이 나오면 동등한 수준으로 가중치 부여.
    - 상위 3개 주요 토픽에 대해 3,000~5,000자의 심층 분석 수행.
